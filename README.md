# Caracoles-patagonicos
Página web de artesanías hechas a mano en la Patagonia, Argentina.
## Sobre el proyecto
Este proyecto es una página web creada con HTML y CSS para mostrar productos artesanales.
## Tecnologías
- HTML
- CSS
## Objetivo
Aprender desarrollo web y crear un sitio real para mostrar productos 100% artesanales.
